Custom fields management

Not enough fields? You can manage (add, modify, delete) one or several custom fields in current catalog.

The custom fields are visibles on right panel in the second tab 'Custom fields'. You can add, modify, delete, move and resize custom fields directly in this tab by a right click on it. You can also click on Manage fields icon, in tools menu.

With custom fields manager window, you can manage all your fields.

To add a new field:

  • Select field '--- New Field ---'.
  • Enter a field tag.
  • Click on Add field icon.

If you want to change field tag:

  • Enter a new field tag.
  • Click on Change tag icon.

The field tag can only containts 'a'-'z', 'A'-'Z', '0'-'9', '_' characteres and can not start with a digit '0'-'9'. Also, some tags are reserved for movie fields (Number, OriginalTitle, TranslatedTitle, ...).

When a field is selected, you can specify its properties:

  • Name: The field name.
  • Extension: The field extension, used to give a precision on field value (e.g. fps, min., MB).
  • Type: The field type.
  • Default value: Value to add to this field when a new movie is created.
  • Import media info: Allows you to define information to extract from media for this field when media file importation.
  • Multiples values: Provides the possibility for a field to contain multiple values separated by a separator (',' by default).
    When grouping on this field, each value will be used to create different groups rather than creating a single group with all values.
    • If the checkbox "Del. parentheses" is checked, then the values in parentheses will be removed for the group name (e.g. Leonardo DiCaprio (Frank Abagnale Jr.) -> Leonardo DiCaprio).
    • If the checkbox "Patch" is checked, then the first parenthesis before values such as (I (...), (II (...), (XV (...), ..., (XXX (...) will be not take into account.
  • List values: Allows you to specify the content of the drop-down list for a field type 'List'.
    • Auto-sort items: Sort list contents alphabetically
    • Auto-complete while typing: If enabled, when you type text in a drop-down list in the main window there will be a "suggestion" of a matching value, like when you type an address in a web browser. This feature may cause problems if you have several items that are similar.
    • Use values of current catalog instead of predefined list: Does not use the values entered in the predefined list, but searchs in the current catalog for values to display in the drop-down list.
    • Automatically add values that are not in list: If the contents of a field is not in the list of this field, the value will be added to the list (e.g. if you enter a country that is not yet in the list, the program will add it to the list).

You can delete selected field by clicking on Delete field icon. You can also delete all fields by clicking on Clear all fields icon.

You can export fields of current catalog by clicking on Export fields icon. In the same way, you can import fields from exported fields or other catalog (.amc, .xml) by clicking on Import fields icon.

About the moving of fields, by pressing the key CTRL, you can move all the fields located below or at the same level of the selected field. This can be very useful to insert a new field between existing fields when there are already a lot of fields.
