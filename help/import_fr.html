Importation depuis un autre format

Vous pouvez importer des données dans la liste de films depuis un autre format de fichier.

Sélectionnez un format dans la liste de la partie gauche de la fenêtre d'importation, et ensuite sélectionnez le fichier source.

Formats disponibles :

  • CSV (peut être utilisé pour importer de Microsoft Excel ou d'autres programmes)
  • Ant Movie Catalog
  • Base de données Microsoft Access
  • DVD Profiler (sur base de son export XML)
  • GCStar (gcstar.org)
  • Fichiers vidéos (noms et informations)

L'option "Permettre les doublons dans les n°" peut être utilisée si vous voulez être sûr que les films importés conservent leur numéro. Lors de l'importation d'un film, si cette option n'est pas activée et qu'il y a déjà un film avec le même numéro, le nouveau film importé recevra un nouveau numéro.
