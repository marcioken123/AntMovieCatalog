Barres d'outils

Tous les éléments du menus sont disponibles en tant qu'icônes pour barres d'outils. Par défaut, le programme affiche les fonctions les plus courantes dans la barre d'outils. Vous pouvez modifier cela via la fenêtre des Options.

Fichier

Nouveau : Crée un nouveau catalogue.

Ouvrir : Charge un catalogue. Il y a deux icônes "Ouvrir". La seconde affiche une petite flèche à sa droite, permettant d'afficher la liste des fichiers ouverts récemment.

Sauver : Sauve le catalogue sous le nom de fichier courant.

Sauver sous : Sauve le catalogue sous un nouveau nom.

Imprimer : Affiche la fenêtre de prévisualisation des rapports.

Importer : Permet d'importer des enregistrements d'autres formats de fichiers.

Exporter : Permet d'exporter le catalogue vers d'autres formats de fichiers.

Propriétés : Affiche des informations sur le fichier courant et sur le catalogue.

Quitter : Quitte le programme.

Film

Ajouter : Ajoute un film dans la liste.

Changer le numéro : Permet de modifier le numéro du film sélectionné.

Supprimer : Supprime le film courant.

Rechercher : Cherche un film sur la base d'un champ spécifié.

Random : Sélectionner un film de manière aléatoire.

Afficher l'image : Affiche ou cache la fenêtre d'image.

Fonctions d'image : Affiche un sous-menu avec les icônes suivantes...

Sélectionner : Sélectionne une image qui sera stockée ou liée au catalogue et associée à ce film.

Sauver l'image : Sauve une copie de l'image dans un fichier (uniquement pour les images stockées, pas encore disponible pour les images liées).

Copier : Copie l'image dans les presse-papiers au format bitmap.

Supprimer : Supprimer l'image du film courant.

Dock/Undock : Change le style de la fenêtre d'image, "docké" (intégré à la fenêtre principale) ou "undocké" (fenêtre flottante).

Afficher les extras : Affiche ou cache la fenêtre des extras.

Fonctions sur l'adresse web : Affiche un sous-menu avec les icônes suivantes...

Ouvrir l'adresse : Ouvre l'adresse ou le fichier spécifié dans le champ.

Ouvrir le dossier : Ouvrir le dossier contenant le fichier spécifié dans le champ.

Copier l'adresse : Copie l'adresse ou le fichier spécifié dans le champ.

Parcourir... : Sélectionner un fichier à renseigner dans le champ.

Fonctions sur le chemin du fichier : Affiche un sous-menu avec les icônes suivantes...

Ouvrir l'adresse : Ouvre l'adresse ou le fichier spécifié dans le champ.

Ouvrir le dossier : Ouvrir le dossier contenant le fichier spécifié dans le champ.

Copier l'adresse : Copie l'adresse ou le fichier spécifié dans le champ.

Parcourir... : Sélectionner un fichier à renseigner dans le champ.

Copier : Copie toutes les informations du film courant vers le presse-papiers.

Coller : Utilises les informations stockées dans le presse-papiers pour le film courant.

Annuler : Recharge les information du film courant. Ces informations sont sauvées lorsque le catalogue est sauvé ou lorsque l'on sélectionne un autre film.

Extraire des infos : Affiche un sous-menu avec les fonctions d'importation suivantes...

De fichiers : Permet de sélectionner des fichiers et d'en extraire des informations (fichiers vidéo ou images). Le résultat est le même que lorsque l'on "drag & drop" des fichiers sur la fenêtre principale.

D'un script : Affiche la fenêtre de script, permettant d'utiliser un fichier de script pour importer des informations de film via internet..

Rechercher sur internet : Affiche un menu (personnalisable par l'utilisateur) permettant de lancer le navigateur web pour rechercher des informations d'un film sur internet.

Affichage

Affichage trié : Trie les films de la liste d'après la valeur d'un champ spécifié.

Affichage groupé : Groupe les films dans la liste d'après la valeur d'un champ spécifié.

Affichage en tableau : Affiche la liste de films sous forme d'une large liste avec les films et toutes leurs informations.

Champs du tableau : Sélectionner les champs en mode tableau à afficher ou cacher.

Affichage en miniatures : Affiche la liste de films sous forme de miniatures.

Etirer la liste : Etirer la liste vers la droite.

Affichage HTML : Affiche les informations du film en fonction du template HTML sélectionné dans les options.

Sélectionner HTML : Sélectionner un template HTML.

Outils

Gérer les champs : Gérer les champs personnalisés du catalogue courant.

Scripting : Affiche la fenêtre de script, permettant d'appliquer un script à une liste de films.

Renuméroter : Renumérote les films, en commençant à 1 avec des chiffres continus. Ceci vous permet de combler les trous apparaissant lorsque vous supprimez des films si vous n'avez pas besoin de conserver des numéros fixes pour chaque film.

Prêts : Liste de tous les films actuellement en prêt, et qui a quel film.

Statistiques : Affiche des graphiques à propos du catalogue.

Options : Permet de modifier les options du programme.

Langue : Change la langue des fenêtres du programme.

Aide

Aide : Affiche ce fichier d'aide.

À propos de : Affiche des informations à propos de l'auteur du programme et la version.
