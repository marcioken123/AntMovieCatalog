Gestion des champs personnalisés

Pas assez de champs ? Vous pouvez gérer (ajouter, modifier ou supprimer) un ou plusieurs champs personnalisés dans le catalogue courant.

Les champs personnalisés sont visibles sur le panneau de droite dans le deuxième onglet "Champs personnalisés". Vous pouvez ajouter, modifier, supprimer, déplacer et redimensionner les champs personnalisés directement dans cet onglet par un clic droit sur le champs personnalisé. Vous pouvez aussi cliquer sur l'icône Gérer les champs, dans le menu outils.

Avec la fenêtre de gestion des champs personnalisés, vous pouvez gérer l'ensemble de vos champs.

Pour ajouter un nouveau champ :

  • Sélectionnez le champ '--- Nouveau champ ---'.
  • Entrez un tag pour le champ.
  • Cliquez sur l'icône Ajouter champ.

Si vous souhaitez changer le tag :

  • Entrez un nouveau tag pour le champ.
  • Cliquez sur l'icône Changer tag.

Le tag du champ peut seulement contenir les caractères 'a'-'z', 'A'-'Z', '0'-'9', '_' et ne peut pas commencer par un chiffre '0'-'9'. Aussi, certains tags sont réservés pour les champs film (Number, OriginalTitle, TranslatedTitle, ...).

Lorsqu'un champ est sélectionné, vous pouvez spécifier ses propriétés:

  • Nom : Le nom du champ.
  • Extension : L'extension du champ, utilisé pour donner une précision sur la valeur du champ (ex: img/s, min., Mo).
  • Type : Le type du champ.
  • Valeur par défaut : La valeur à ajouter au champ lors de la création d'un nouveau film.
  • Importer info vidéos : Permet de définir l'information à extraire des vidéos pour ce champ lors de l'importation de vidéos.
  • Valeurs multiples : Donne la possibilité à un champ de contenir plusieurs valeurs séparés par un séparateur (',' par défaut).
    Lors du groupage sur ce champ, chaque valeur sera utilisée pour créer des groupes différents plutôt que de créer un seul groupe contenant toutes les valeurs.
    • Si la case "suppr. parenthèses" est coché, alors les valeurs entre parenthèses seront supprimées pour le nom du groupe (ex: Leonardo DiCaprio (Frank Abagnale Jr.) -> Leonardo DiCaprio).
    • Si le "patch" est coché, alors la première parenthèse devant des valeurs comme (I (...), (II (...), (XV (...), ..., (XXX (...) ne sera pas prises en compte.
  • Liste de valeurs : Permet de spécifier le contenu de la liste déroulante pour un champ de type 'Liste'.
    • Tri automatique : Trie par ordre alphabétique le contenu de la liste.
    • Auto-complétion lors de la frappe : Si activé, lorsque vous tapez du texte dans une liste déroulante dans la fenêtre principale il y aura une "suggestion" de la valeur dont le début correspond à ce que vous avez tapé, comme lorsque vous tapez une adresse dans un navigateur web. Cela peut causer des problèmes si vous avez plusieurs éléments similaires.
    • Utiliser les valeurs du catalogue courant plutôt que la liste prédéfinie : N'utilise pas les valeurs entrées dans la liste prédéfinie, mais cherche dans le catalogue courant les valeurs à afficher dans la liste déroulante.
    • Ajouter automatiquement les valeurs qui ne sont pas dans la liste : Si le contenu d'un champ n'est pas dans la liste déroulante de ce champ, la valeur sera ajoutée à la liste (par exemple si vous entrez un pays qui n'est pas dans la liste le programme va l'ajouter à la liste).

Vous pouvez supprimer le champ sélectionné en cliquant sur l'icône Supprimer champ. Vous pouvez aussi effacer tout les champs en cliquant sur l'icône Effacer tout les champs.

Vous pouvez exporter les champs du catalogue courant en cliquant sur l'icône Exporter champs. De même, vous pouvez importer des champs à partir de champs exportés ou d'un autre catalogue (.amc, .xml) en cliquant sur l'icône Importer champs.

Concernant le déplacement des champs, en pressant la touche CTRL, il est possible de déplacer tous les champs se trouvant en dessous ou au même niveau que le champ sélectionné. Ceci peut être très utile pour insérer un nouveau champ entre les champs existants lorsqu'il y a déjà beaucoup de champs.
