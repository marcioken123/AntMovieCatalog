Informations d'un film

Dans la fenêtre principale vous pouvez entrer pas mal d'informations pour chacun des films :

  • Label du support : Vous pouvez indiquer ici le numéro du disque, de la cassette, le code-barre, un label, ou n'importe quoi d'autre
  • Type de support : Le support utilisé pour stocker le film (BR, DVD, CD, VHS, etc...)
  • Origine : Où/Comment vous avez eu ce film
  • Date d'ajout : Date à laquelle le film a été ajouté à la liste
  • Emprunté par : Nom de la personne qui a emprunté le film
  • Date vu : Date à laquelle le film a été visionné pour la dernière fois
  • Ma note : Avez-vous aimé le film ? Vous pouvez le dire ici
  • Note : Autre note de la presse ou des utilisateurs
  • Titre original
  • Titre traduit
  • Réalisateur
  • Producteur
  • Scénariste
  • Compositeur
  • Acteurs
  • Pays
  • Année
  • Durée : Longueur du film, en minutes
  • Catégorie
  • Certification : Public visé / Age minimum conseillé pour regarder le film
  • Adresse web : Vous pouvez indiquer ici une adresse de site web en relation avec le film. Vous pouvez lancer le navigateur par défaut pour ouvrir cette URL en cliquant sur le bouton à droite du champ
  • Description
  • Commentaires
  • Chemin fichier : Chemin du fichier vidéo (même fonctionnement que le champ Adresse web)
  • Format vidéo : Codec ou média utilisé pour la vidéo
  • Débit vidéo : Débit de la vidéo en kilobits par seconde
  • Format audio : Codec ou média utilisé pour l'audio
  • Débit audio : Débit de l'audio en kilobits par seconde
  • Résolution : Largeur et hauteur du film (ex. 640x480)
  • Fréquence : Nombre d'images par seconde
  • Taille des fichiers : Taille en méga-octets du film. Il peut y avoir plusieurs fichiers (ex. 650+590 ou 1240)
  • Nombre de disques : Nombre de disques ou de cassettes
  • Langues
  • Sous-titres

Toutes les listes déroulantes stockent et récupèrent leurs valeurs du fichier XML où sont stockées les préférences. Vous pouvez les modifier dans la fenêtre des Options dans la rubrique "Listes déroulantes". Une option permet d'utiliser les valeurs du catalogue courant plutôt que les listes prédéfinies.

Il y a d'autres "champs" additionnels qui apparaissent dans certaines fenêtres :

  • Coché : L'état des cases à cocher dans la liste principale ; elles peuvent être activées via les Options si vous ne les voyez pas
  • Etiquette couleur : Les étiquettes couleurs associées aux films dans la liste principale (case à cocher de couleur et/ou ligne de couleur) ; elles peuvent être activées via les Options si vous ne voyez aucune couleur (blanc par défaut)
  • Titre formaté : Titre utilisé dans la liste principale, qui peut combiner à la fois le titre original et le titre traduit par exemple ; consultez la page des Options pour plus d'informations
  • Etat de l'image: Retourne l'état de l'image du film (stockée dans le catalogue, copiée, externe, ...)
  • Nombre d'extras: Retourne le nombre d'extras du film
