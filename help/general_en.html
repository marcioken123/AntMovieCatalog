General information

This program allows to manage catalog of movies. This mean that you can have several catalogues to classify movies separately. Each catalog can contain a nearly-infinite number of movies, and is stored in a file that can have a ".amc" or a ".xml" extension depending of the selected format. These cataloges can be managed with the New, Open, Save and Save As icons of the toolbar and File menu.

When a catalog is open (or in an empty new one), you can add new movies with the Add icon that will display the number select window (each movie has to have a number, but you can ask to use automatic numbering by checking the "Do not ask again" option of this window). Using this command, you can add movies to the catalog, do not forget to save it sometimes.

Now you can fill-in the various fields. The Get Information icon displays a menu that allows you to import information from files or from script.

  • The importation from files will try to get all useful information from media files (.avi, .mpeg, etc.). you can also drag & drop a file from Windows Explorer to Ant Movie Catalog window.
  • The importation from script displays the Scripting window that provides scripts for importing information from various websites. This is the easiest way to fill all the movie-related fields if you have an Internet connecion.

You can also store or link a picture to your catalog. Some scripts provide a picture when importing information, but you can also manually add a picture from your computer by clicking the Select Picture icon (located at the bottom left corner of the main window) or by doing a drag & drop of a picture from Windows Explorer.

If you want to add new fields, you can manage custom fields of current catalog with the Manage fields icon in tools menu.

You can also add extras for each movie. Each extra can have a title, a description, an url, a picture, ... and can belongs to one or more categories. Then you can add all extras you need for a movie such as covers, fanarts, snapshots, casting, trailers, videos, ...

If you have your list in another format (e.g. a Microsoft Excel document) you can use the Import window to import it into the current catalog.

Do not hesitate to discover all the functionalities available in the menus. Take also some time to inspect all available options available in the Preferences window through the Preferences icon.
