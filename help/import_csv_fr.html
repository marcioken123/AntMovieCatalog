Importation CSV

CSV est un format de fichier texte qui est très utile du fait que beaucoup de programme permettent d'ouvrir et enregistrer ce type de fichiers (par exemple Microsoft Excel).

Dans ces fichiers chacun des enregistrements est sur une ligne et les champs sont séparés par un caractère (généralement une virgule ou un point-virgule).

Quand vous importez ce type de fichiers, chaque ligne sera considérée comme un film dans la liste. Ensuite la ligne est séparée en plusieurs champs. Le programme ne peut pas deviner à quel champ du film correspond chacune des colonnes des enregistrements CSV. Vous allez donc devoir le lui expliquer.

Pour cela vous devez cliquer sur l'en-tête de colonne pour l'associer à un champ du catalogue.

Répétez cette étape pour chacune des colonnes qui doit être importée.

Si vous n'associez pas de champ à une colonne, cette colonne ne sera pas utilisée pour l'importation de données.

Si aucune colonne "Numéro" n'est spécifiée, les numéros seront créés à partir du plus haut numéro existant + 1. Idem pour les numéros provenant du fichier CSV et étant déjà présents dans la liste, donc il n'y a pas de numéros en double.

Les champs de type date doivent être au format 'yyyy-mm-dd' ou régional pour être lus correctement.
