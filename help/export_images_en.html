Export images only

This allows you to extract all the images from the catalog.

You can specify easily which names the images will have by selecting an element in each of the three lists:

  • The prefix to use (none, catalog name, exported filename)
  • + The information of movie to use (number, formatted title, ...)
  • + The information of extra to use (number, title, ...) for extra pictures
You can also change the separator between values if you want.
