Import from another format

You can import data to the movie list from other file format.

Select one of the format in the list that is one the left of the "Import" window, then select the source file.

Available file formats:

  • CSV (can be used to import from Microsoft Excel and other programs)
  • Ant Movie Catalog
  • Microsoft Access Database
  • DVD Profiler (from its XML export)
  • GCStar (gcstar.org)
  • Media files (names and information)

The "Allow duplicate numbers" option can be used if you want to be sure that the movies that are imported keep their number. When importing a movie, if this option is not enabled and that there is already a movie with the same number, the new imported movie will receive a new number.
