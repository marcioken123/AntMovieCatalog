Impression

Dans cette fenêtre vous pouvez prévisualiser les rapports avant leur impression.

Sélectionnez un modèle dans la liste de gauche pour le prévisualiser. Cette liste propose tous les fichiers trouvés dans le sous-dossier "Templates".

Si vous voulez ouvrir un modèle présent dans un autre dossier, vous pouvez utiliser le bouton "Charger un modèle".

Vous pouvez spécifier un ordre de tri : par numéro, par titre original, par titre traduit, ou choisir les options de tri avancées. Lorsque vous sélectionnez cette option, une fenêtre apparaît et vous permet de choisir quel champ doit être inclus dans le tri, et dans quel ordre.

Les options "Films à inclure" permettent de restreindre l'impression aux films sélectionnés, cochés (lié à l'option "Cases à cocher") ou visibles (lors de l'utilisation de l'option "Afficher les résultats" de la fonction de recherche de la fenêtre principale).

Lorsque vous êtes prêts à imprimer, cliquez sur le bouton "Imprimer". Une fenêtre de confirmation s'affiche alors, permettant de sélectionner l'imprimante à utiliser.

Note importante : Si vous voulez utiliser une autre taille de papier, vous devez modifier le modèle avec le Designer (par défaut la taille de papier est généralement A4, mais en fait ça dépend de qui a fait le modèle).

Si vous désirer modifier un modèle ou en crée vous-même, cliquez sur le bouton "Report Designer". Pour obtenir de l'aide sur son utilisation, consultez la page Création de rapports dans la section consacrée aux Informations techniques.
