Films prêtés

Ce module vous permet de gérer les films que vous amis vous empruntent.

La liste dans la moitié supérieur de la fenêtre contient tous les films du catalogue courant. Dans le coin inférieur gauche vous trouvez la liste des noms des emprunteurs. Dans le coin inférieur droit vous trouvez la liste des films qui sont actuellement empruntés par la personne sélectionnée (dans la liste à gauche).

Pour marquer un film comme "Emprunté", vous devez sélectionner le film, ainsi qu'un nom dans la liste des emprunteurs. Ensuite vous pouvez utiliser le bouton "Sortir", ou double-cliquer sur le film.

Pour retirer le status "Emprunté" d'un film, sélectionnez-le dans la liste de l'emprunteur, et cliquez sur le bouton "Rentrer" ou double-cliquez sur le titre du film.

Vous pouvez sélectionner plusieurs films à la fois à l'aide des touches Ctrl et Shift.

Si vous utilisez un numéro unique ou le champ "Label" pour indiquer que plusieurs films se trouvent sur le même support, vous pourrez avoir besoin des options "Inclure les films ayant le même numéro" et "Inclure les films ayant le même label" accessibles via l'icône "Options". Lorsque ces options sont activées, les films ayant le même numéro ou label que les films sélectionnés sont prêtés à la même personne (à moins qu'ils soient déjà assignés à quelqu'un d'autre).

Toutes les actions (entrées / sorties) sont enregistrées dans un fichier texte délimité par des tabulations. Consultez la page consacrée aux Options pour plus d'informations.
