Movie information

In the main window you can enter several information for each movie:

  • Media label: You can store here the number of the disc, tape, barcode, label or anything else
  • Media type: The support used to store movie (BR, DVD, CD, VHS, etc...)
  • Source: Where/How you got the movie
  • Date added: When the movie was added to the list
  • Borrower: Name of the friend that borrowed the movie
  • Date watched: When the movie has been watched for the last time
  • My rating: Did you like this movie ? You can say it here
  • Rating: Other rating from press or users
  • Original title
  • Translated title
  • Director
  • Producer
  • Writer
  • Composer
  • Actors
  • Country
  • Year
  • Category
  • Certification : Intended audience / Minimum age advisable to watch the movie
  • Length: Length of the movie, in minutes
  • URL: You can enter here a website address in relation with the movie. You can launch the default browser to surf to this URL by clicking on the button that is at the right of the field
  • Description
  • Comments
  • File path : File path of video file (same behaviour than field URL)
  • Video format: Codec or media used for video
  • Video bitrate: Video bitrate in kilobits per second
  • Audio Format: Codec or media used for audio
  • Audio bitrate: Audio bitrate in kilobits per second
  • Resolution: Width and Height of the movie (e.g. 640x480)
  • Framerate: Number of images per second
  • File Sizes: Size in megabytes of the movie. There can be multiple files (e.g. 650+590 or 1240)
  • Number of disks: Number of disks or tapes
  • Languages
  • Subtitles

All the lists store and retrieve their values from the XML file where are stored the preferences. You can edit them in the "Drop-down lists" page of the Preferences window. There is also an option to use current catalog values rather than predefined lists.

There are others additional "fields" that appear in some windows:

  • Checked: The state of the checkboxes in the main list; they can be enabled in the Preferences if you do not see them
  • Color Tag: The color tags associated to the movies in the main list (checkboxe color and/or line color); they can be enabled in the Preferences if you do not see any color (white by default)
  • Formatted title: Title displayed in the main list, that can combine both original and translated title for example; see Preferences for more information
  • Picture status: Returns the status of movie picture (stored in catalog, copied, external, ...)
  • Number of extras: Returns the number of movie extras
