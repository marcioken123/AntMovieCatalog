About the author, credits & links

This program has been made by Antoine Potten and Mickaël Vanneufville with Borland Delphi 7 under Windows 10.

It has been designed to run on Windows and has been tested on Windows 7, 8 and 10. It may run on older versions of Windows, but has not been tested recently on these.

Initially the program has been created by Antoine Potten because he had no program that can manage its list of movies and easily create an HTML page with the list. Ten years later, the development has been resumed by Mickaël Vanneufville who made all improvements since version 3.6 until 4.2.1.
