Exportation des images uniquement

Ceci vous permet d'extraire les images du catalogue.

Vous pouvez spécifier facilement quels noms auront les images en sélectionnant un élément dans chacune des trois listes :

  • Le préfixe à utiliser (aucun, nom du catalogue, nom du fichier exporté)
  • + L'information du film à utiliser (numéro, titre formaté, ...)
  • + L'information de l'extra à utiliser (numéro, titre, ...) pour les images des extras
Vous pouvez aussi changer le séparateur entre les valeurs si vous le souhaitez.
