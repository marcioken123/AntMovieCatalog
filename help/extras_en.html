Extras management

Movie fields and custom fields are not enough for you? You can also add one or more extras for each movie.

The extras are visible on right panel in the last tab 'Extras'. You can also display them to the right permanently when you are not on tab 'Extras' by clicking on Show or hide extras in main toolbar or menu. You can import (shortcut Ctrl+Ins), add (shortcut Ins), edit (shortcut M) and delete extras (shortcut Del) thanks to extras toolbar or by a right click in extras window.

In window to edit extras , you can easily modify fields and picture of one or more selected extras in the same time.

Each extra has the following fields:

  • Number: Extra number between 1 and the number of extras. This number matches to the position of the extra in the extra list of movie.
  • Tag: Extra tag. This field is mainly used for HTML export and printing to identify an extra of a movie but there is no restriction in its use.
  • Title: Extra title.
  • Category: Extra category (e.g. covers, fanarts, snapshots, casting, trailers, videos, ...).
  • URL: URL or file path of associated extra.
  • Description: Extra description.
  • Comments: Extra comments.
  • Created by: Extra creator. This field is mainly used for scripts in order to know what extras have been created by themselves and so update them easier without modify the others.
  • Picture: Extra picture.

Once you have created your extras, you can easily see them in extras window. You can show picture of selected extra by clicking on Show picture (shortcut P) and access to the associated web page or file by clicking on URL (shortcuts U, D, C, B).

You can also copy / paste several extras in the same time by clicking on Copy and Paste (shortcuts Ctrl+C, Ctrl+V).

If you need, you can also renumber all extras of a movie or several movies by clicking on Renumber.
