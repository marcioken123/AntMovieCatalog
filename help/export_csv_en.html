Export to CSV

CSV is a text file format which is really useful since many program allow to load and save such files (e.g. Microsoft Excel).

In these files each item is on one line, and all the fields of an item are separated by a character (usually comma or semicolon).

Date fields will be formatted as 'yyyy-mm-dd'.

If you enable the picture export, a "Picture" column will be added, containing the file name of the picture.

If you export an extra field, the values of the same field of each extra will be concatenated and separated by <+> by default.

If you enable the export of extra pictures, a "EPicture" column will be added, containing the file names of the pictures separated by <+> by default.
