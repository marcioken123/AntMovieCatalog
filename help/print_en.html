Printing

In this window you can preview reports before printing them.

Select a template in the list of the left to preview it. In this list you see the files that were found in the "Templates" subfolder.

If you want to open a template from another folder you can use the "Load template" button.

You can specify a sort order: by number, original title, translated title, or select advanced sort options. When you select this option a window appears and you can select which field should be includ for sort, and in which order.

The "Movies to include" options allows to restrict the exportation to selected movies, checked movies (that is linked to the "Checkboxes" option) or visible movies (when using the "Display results" option of main window's search function).

When you are ready to print, simply click on the Print button, then a confirmation dialog will allow you to select the printer to use.

Important note: If you want to use another paper size, you have to modify the template with the Report Designer (default paper size is usually A4, but it depends of who made the template actually).

If you want to modify or create your own templates, click the "Report Designer" icon. For help on how to use it, see the Reports creation page in the Technical information section.
