Informations générales

Ce programme permet de gérer des catalogues de films. Cela signifie que vous pouvez avoir plusieurs catalogues pour classer des films séparément. Chaque catalogue peut contenir un nombre quasi infini de films, et est stocké dans un fichier ayant une extension ".amc" ou ".xml" selon le format choisi. Ces catalogues peuvent être gérés via les icônes Nouveau, Ouvrir, Sauver et Sauver sous de la barre d'outils et du menu Fichier.

Lorsqu'un catalogue est ouvert (ou qu'il est vide), vous pouvez ajouter de nouveaux films via l'icône Ajouter qui affichera la fenêtre de sélection de numéro (chaque film doit avoir un numéro, mais vous pouvez cocher la case "Ne plus afficher" afin d'utiliser la numérotation automatique). À l'aide de cette commande, vous pouvez ajouter vos films au catalogue, n'oubliez pas de le sauver de temps en temps.

Vous pouvez maintenant remplir les différents champs. L'icône Extraction d'information affiche un menu permettant d'importer des informations de fichiers ou d'un script.

  • L'importation de fichiers tentera de prendre toutes les informations utiles dans les fichiers multimédia (.avi, .mpeg, etc.). Vous pouvez aussi effectuer un glisser-déposer (drag & drop) des fichiers depuis l'Explorateur Windows vers la fenêtre de Ant Movie Catalog.
  • L'importation d'un script affiche la fenêtre Scripting proposant des scripts pour l'importation d'informations depuis divers sites web. Il s'agit de la manière la plus simple de remplir les différents champs relatifs aux films si vous disposez d'une connexion à Internet.

Vous pouvez aussi stocker ou lier une image au catalogue. Certains scripts fournissent une image lors de l'importation des informations, mais vous pouvez également manuellement ajouter une image stockée dans votre ordinateur en cliquant sur l'icône Sélectionner une image (située en bas à gauche de la fenêtre principale) ou via un glisser-déposer de l'image depuis l'Explorateur Windows.

Si vous souhaitez ajouter de nouveaux champs, vous pouvez gérer les champs personnalisés du catalogue courant via l'icône Gérer les champs dans le menu outils.

Vous pouvez aussi ajouter des suppléments pour chaque film que l'on nommera extras. Chaque extra peut contenir un titre, une description, une adresse, une image, ... et peut appartenir à une ou plusieurs catégories. Vous pouvez donc ajouter tous les suppléments que vous souhaitez pour un film comme des affiches, fanarts, images du film, casting, bandes-annonces, vidéos, ...

Si votre liste actuelle est dans un autre format (par exemple un document Microsoft Excel) vous pouvez utiliser la fenêtre d'Importation pour la transférer dans le catalogue courant.

N'hésitez pas à découvrir toutes les fonctionnalités offertes dans les menus. Prenez aussi le temps d'inspecter toutes les réglages disponibles dans la fenêtre des Options via l'icône Options.
