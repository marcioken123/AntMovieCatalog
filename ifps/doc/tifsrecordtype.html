PIFSRecordType

PIFSRecordType is used to store the definition of a record

Type:
PIFSRecordType = ^TIFSRecordType;
TIFSRecordType = packed record
  u: string; // stored like 'Name1 Type1 Name2 Type2' Types as pointers casted to longints.
end;
