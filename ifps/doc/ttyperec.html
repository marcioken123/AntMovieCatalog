TTypeRec

TTypeRec is used to store all type definition.

Type:
PTypeRec = ^TTypeRec;
TTypeRec = record
  Ident: string;
  atypeid: Word;
  ext: Pointer;
end;

Ext can be of type:
CSV_Record: PIFSRecordType
CSV_Array: PTypeRec
CSV_Class: PIFSClassType
CSV_ClassRef: PTypeRec
CSV_Property: PTypeRec
CSV_TypeCopy: PTypeRec
CSV_ProcVar: PIfsProcType
CSV_ExternalObject: PIFSCustomObjectType
